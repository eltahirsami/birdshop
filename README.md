# 🧾 Smart POS System

A complete **Point of Sale (POS)** and inventory management system for stores and businesses.

---

## 🚀 Features
- Product management (Add / Edit / Delete)
- Inventory tracking
- Sales & invoices
- User roles (Admin / Cashier)
- Dashboard analytics
